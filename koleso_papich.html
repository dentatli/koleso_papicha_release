Создайте OAuth-приложение

Чтобы сайт мог получать донаты, создайте OAuth-приложение DonationAlerts.

Для подключения DonationAlerts запустите сайт через start_koleso.bat
Открыть DonationAlerts

Скопируйте этот адрес и вставьте его в поле URL перенаправления при создании приложения DonationAlerts.

  1. Имя приложения: любое, например «колесо».
  2. URL перенаправления: вставьте адрес из поля выше.
  3. Нажмите «Создать».
  4. Скопируйте ID приложения и API Key.
Не показывайте страницу подключения на стриме. Токен может кратко появиться в адресной строке, после чего сайт автоматически его скрывает.

Тестовый донат

Позволяет проверить очередь донатов без реального платежа
Источник: Тестовый источник

Как проверить

  1. Добавьте лоты во вкладке «Аукцион».
  2. Создайте тестовый донат здесь.
  3. Вернитесь в «Аукцион» → «Донаты» → «В ожидании».
  4. Назначьте донат нужному лоту одним кликом.

Настройки сервисов и история донатов сохраняются вместе с остальным состоянием приложения.
