Расширенные настройки

После авторизации локальный PowerShell-сервер периодически запрашивает новые донаты и передаёт их приложению.

AI настройка

Подключить OpenRouter

Вставьте API key. Он сохранится только локально в server-side secrets.json через DPAPI и не попадёт в браузерный localStorage.

НЕ ПОКАЗЫВАЙТЕ API KEY НА СТРИМЕ.
Открыть страницу ключей OpenRouter
  1. Откройте OpenRouter и создайте API key.
  2. Скопируйте ключ и вставьте его ниже.
  3. После сохранения поле очистится, а в настройках появится статус “Подключено”.
Опционально

Proxy для OpenRouter

Используйте HTTP proxy, если local_server.ps1 не может достучаться до openrouter.ai напрямую.

  1. Запустите ваш proxy/VPN bridge.
  2. Вставьте локальный HTTP proxy URL, например http://127.0.0.1:7890.
  3. Нажмите “Сохранить”, затем “Проверить” в настройках AI.

SOCKS5 напрямую Windows PowerShell не поддерживает. Нужен HTTP proxy.

Тестовый донат

Позволяет проверить очередь донатов без реального платежа
Источник: Тестовый источник

Как проверить

  1. Добавьте лоты во вкладке «Аукцион».
  2. Создайте тестовый донат здесь.
  3. Вернитесь в «Аукцион» → «Донаты» → «В ожидании».
  4. Назначьте донат нужному лоту одним кликом.

Настройки сервисов и история донатов сохраняются вместе с остальным состоянием приложения.
