# Колесо для аукциона донатов

Локальный сайт для ведения аукциона лотов и прокрутки колеса на выбывание.

## Запуск

1. Скачайте файлы проекта.
2. Запустите `start_koleso.bat`.
3. Откроется локальная страница колеса.

## Файлы

- `koleso_papich.html` — основной интерфейс.
- `local_server.ps1` — локальный сервер.
- `start_koleso.bat` — запуск проекта.

## Важно

Проект хранит настройки локально в браузере.
Не показывайте вкладку админки и настройки донат-сервисов на стриме.
